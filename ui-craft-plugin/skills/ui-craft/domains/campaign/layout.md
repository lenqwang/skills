# Campaign 布局

> 活动页布局定义，从 core layout 中拆出的活动线专属部分。
> 通用栅格系统和响应式策略见 core 层。

---

## 页面类型

### campaign.home — 活动首页（默认）

```
Hero（标题 + 副标题 + CTA + 时间/倒计时）
    |  80px 上下内边距
友情提示（可选）
    |  var(--space-section)
活动区块 x N
    +-- 区块标题 + 描述
    +-- 奖池/统计信息
    +-- 表格/卡片/排行榜
    |  var(--space-section)
加码奖励（可选）
    |  var(--space-section)
相关产品推荐（可选）
    |  var(--space-section)
活动规则
    |  96px 底部间距
底部 CTA（可选）
```

### campaign.detail — 活动详情页

```
面包屑导航（首页 > 活动中心 > 当前活动）
    |  24px
活动标题区
    +-- 标题（headline.large）
    +-- 状态标签（进行中/已结束）+ 时间范围
    |  var(--space-section)
活动内容区（单列）
    +-- 富文本 / 规则说明 / 奖品详情
    |  var(--space-section)
相关推荐（可选）
    +-- 相关活动卡片（3 列网格）
    |  var(--space-section)
活动规则（折叠面板）
    |  96px 底部间距
底部 CTA（可选）
```

## 模块骨架

### Hero 区

```
+-----------------------------------------------------+
|                    Hero Section                      |
|  +------------------------+  +--------------------+  |
|  |                        |  |                    |  |
|  |    文案区 (60%)         |  |    插图区 (40%)     |  |
|  |    - H1 标题            |  |    - 图片/动画      |  |
|  |    - 副标题             |  |                    |  |
|  |    - CTA 按钮           |  |                    |  |
|  |                        |  |                    |  |
|  +------------------------+  +--------------------+  |
|                                                      |
|  列间距: 48px                                        |
|  内边距: 80px 上下                                    |
+-----------------------------------------------------+
```

### 活动区块（黑卡嵌白卡）

```
+-----------------------------------------------------+
|  黑卡容器（区块标题 + 描述）                          |
|  +--------------------------------------------------+|
|  |  白卡容器（表格/内容）                            ||
|  +--------------------------------------------------+|
+-----------------------------------------------------+
```

## 引用方式

style 文件中声明布局：
- `layout: campaign` — 等同于 `campaign.home`
- `layout: campaign.home + campaign.detail` — 多页面活动
