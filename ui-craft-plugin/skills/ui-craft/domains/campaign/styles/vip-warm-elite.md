## 区块 4: 布局结构

> layout: campaign

### 差异项

- Hero: 数据大屏模式（标题 + 倒计时 + 统计数据行 + CTA），非默认简单 Hero
- 额外模块: 赛季 Tab → 分区 Tab（可选）→ 双维度排行榜 → 奖池阶梯表 → 奖级指示卡横排 → 奖池分配矩阵 → 幸运抽奖池/刮刮乐 → 实物奖品网格 → 任务入口 → 活动规则
- 栅格: 奖级指示卡多列横排（平板/移动端 overflow-x-auto），奖池分配矩阵多列（平板/移动端 overflow-x-auto），实物奖品 4 列（平板 2 列，移动端 1 列）
- 间距: `--space-section` 120px（桌面）/ 64px（移动端），比默认 64px 更大
- 响应式: 赛季 Tab 横向滚动，矩阵表格横向滚动

## 区块 6: 适用性 & 自检

### 自检清单

> 基础清单见 `contract/rules.md` 自检清单章节，此处仅列本风格特有检查项。

- [ ] 主卡片背景用 `var(--bg-card)` (#141414)，不再用 #201D17 作为通用卡片色
- [ ] 排行榜行背景用 `var(--bg-warm)` (#201D17)，边框用 `var(--bg-warm-border)` (#2A281D)，两者不混用
- [ ] 亮黄 `var(--accent-highlight)` (#F0FB4E) 仅用于特效强调场景，不作为常规文字色
- [ ] 金色层级正确：`var(--accent)` (#FFDD9C) 用于 UI 控件 vs #DAA520 用于排名/奖励数字
- [ ] 收益渐变起始色为 #54E623（石灰绿），非旧版 #17E5A1
- [ ] 紫蓝渐变 (#5322E6 -> #2323E6) 仅用于特效按钮/徽章，非通用色
- [ ] 模块间距 `var(--space-section)` 120px/64px（桌面/移动端）
- [ ] 渐变文字使用 `bg-clip-text text-transparent` 实现
- [ ] 赛季 Tab 金色活跃态对比度足够（`var(--accent)` 背景 + 黑色文字）
- [ ] VIP 图标使用 `@gate/iconfont`，不使用 emoji

### 适用 / 不适用场景

**适用：**
- VIP 合约争霸赛（多赛季/多期数）
- VIP 专属交易对决
- 年终/年中盛典活动
- 含刮刮乐/抽奖的激励活动
- 实物奖品 + 数字奖品混合展示
- 分 VIP 等级的赛区制活动

**不适用：**
- 轻量新人任务活动（建议用 default.md）
- 无 VIP 分级的普通竞赛（建议用 competition.md 或 racing-premium.md）
- 纯空投/福利活动（建议用 default.md）
- 节日主题活动（建议用 festive-gamify.md）
- 偏叙事回顾类页面
