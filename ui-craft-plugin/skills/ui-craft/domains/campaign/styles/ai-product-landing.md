## 区块 2: 风格画像

- **视觉情绪**：科技前沿、沉稳专业、开放可信
- **核心组件**：品牌 Hero（超大字 + 标语）、交替图文区块、图标特性网格、蓝色 CTA 横幅、多栏页脚
- **强调方式**：Gate 蓝用于 CTA 按钮和横幅；品牌名使用超大 display 字体制造视觉锚点；"AI" 等关键词蓝色高亮
- **适用标签**：`AI产品` `智能体` `工具落地页` `产品介绍` `技术平台`

| 维度 | 本风格 | 默认 | 说明 |
|------|--------|------|------|
| 色温 | 冷色 | 中性 | 纯黑底 + 蓝色强调，科技冷调 |
| 信息密度 | 中 | 中 | 交替图文 + 特性网格，节奏舒适 |
| 装饰程度 | 克制 | 克制 | UI 截图作为装饰，无多余视觉元素 |

## 区块 4: 布局结构

> layout: portal.index

### 差异项

- Hero: 全宽深色背景 + 居中品牌超大字 + 标语 + CTA 按钮，非卡片容器式 Hero
- 特性区块: 交替左右布局（奇数: 文左图右，偶数: 图左文右），图片侧为 UI mockup 截图
- 特性网格: 2x2 图标网格，每格含图标 + 标题 + 描述
- 导航栏: 二级导航样式（logo + 中间链接组 + 右侧深浅切换 + CTA 按钮）
- CTA 横幅: 全宽蓝色背景横幅，居中标语 + 按钮
- 页脚: 多栏链接布局（品牌列 + 4-5 个链接列 + 版权行）
- 内容宽度: 1200px 居中
- 区块间距: 96px（比默认 64px 更大，营造开阔感）

## 区块 6: 适用性 & 自检

### 自检清单

> 基础清单见 `contract/rules.md` 自检清单章节，此处仅列本风格特有检查项。

- [ ] [B] Hero 品牌名使用超大 display 字体（>= 60px），形成明确视觉锚点
- [ ] [B] 交替图文区块严格执行奇偶交替（文左图右 / 图左文右）
- [ ] [W] UI mockup 容器使用 `var(--bg-elevated)` + `var(--border)` 边框，非纯色块
- [ ] [B] CTA 横幅使用 `var(--cta-banner-bg)` 全宽蓝色背景，非卡片
- [ ] [W] 主标语中关键词（如"AI 智能体"）使用 `var(--brand)` 蓝色高亮
- [ ] [W] 区块间距使用 96px（`var(--space-section)`），营造开阔节奏
- [ ] [W] 图标特性网格为 2 列布局，图标使用 @gate/iconfont 32px 尺寸
- [ ] [B] 页脚多栏链接布局，非单行

### 适用场景

- AI 产品/工具落地页（智能体、工作站、AI 助手）
- 技术平台产品介绍页
- SaaS 产品特性展示页
- 开发者工具产品首页

### 不适用场景

- 交易竞赛活动（建议用 competition.md）
- VIP 专属活动（建议用 vip-warm-elite.md）
- 开发者技能市场/列表页（建议用 dev-portal.md）
- 节日主题活动（建议用对应 festive 系列）
- 数据密集型仪表盘
