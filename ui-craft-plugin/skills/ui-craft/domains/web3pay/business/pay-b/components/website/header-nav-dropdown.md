# Header 导航下拉面板

> 来源: Figma [Pay-B端官网](https://www.figma.com/design/5fRySZo785Q1QHfn8FxqeN/) · Node `8577:49808`
> 归属: web3pay / Pay-B 官网（营销 Web）
> 版本: v1.0.0
> 补充: `header-footer.md` 中 Header 的导航下拉面板详细规格

## 组件概述

Header 导航项（产品/行业方案/开发者）的下拉面板规格。点击或 Hover 导航项时展开白色下拉面板，包含分类导航链接和图标。

---

## Header 状态变体

| 变体 | Figma 标签 | 说明 |
|------|-----------|------|
| **Web_未登录** | `8577:49808` 子节点 | 默认未登录态，显示 Log In + Sign Up |
| **Web_已登录** | `8577:49808` 子节点 | 已登录态，显示用户头像 + 红色通知点 |
| **Web_点击头像後出** | `8577:49808` 子节点 | 头像点击后弹出用户菜单（含头像、用户名、"Log Out" 按钮） |

## 下拉面板容器规格

| 属性 | 值 | 说明 |
|------|-----|------|
| 背景 | `var(--layer/3, white)` | 白色 |
| 圆角 | **8px** | `border-radius: 8px` |
| 阴影 | `0px 6px 16px rgba(0,0,0,0.08)` | Dropdown Shadow |
| 内边距 | **16px** | 四周 |
| 定位 | Header 下方，与触发项左对齐 | absolute |
| z-index | 200 | 浮于 Header 之上 |

## 交互规则

| 规则 | 说明 |
|------|------|
| 触发方式 | Hover 或 Click 导航项 |
| 展开动画 | `opacity 0→1, transform translateY(-8px)→0`, 200ms ease |
| 关闭 | 鼠标离开面板区域 / 点击其他导航项 / 点击面板外 |
| 互斥 | 同一时间只展开一个下拉面板 |
| 箭头 | 展开时 `CEX_down_fill` 旋转 180° |
