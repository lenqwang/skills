## 五、提示文案模板

### 48小时审核提示（强制，审批流相关页面必须展示）

```
为防止审批员发起下发，审批人员需小于角色成员总数，
当前至多可选 N 位。
审核需在订单发起后 48小时内 完成，超时未处理系统将自动拒绝该订单。
```

### 汇率换算提示（下发数量相关必须展示）

```
下发币种非USDT时，将按实时汇率换算成USDT。
```

### 规则数量上限提示

```
当前规则数量已达到最大上限100条，请删除不需要的规则后再新建。
```

### 无匹配规则说明

```
当下发订单在上方列表中找不到匹配规则时，系统将采取此默认动作（配置后对主账户及子账户有效）。
```

## 七、弹窗文案模板

### 二次确认弹窗

```
标题：是否确认{动作}当前规则「{规则名称}」
描述：{动作}后{后果说明}。确认后需安全校验。
按钮：取消 / 确认{动作}
```

示例：
```
标题：是否确认删除当前规则「高额单笔拒绝规则」
描述：删除后该规则将不再生效。确认后需安全校验。
按钮：取消 / 确认删除
```

### 安全验证弹窗

```
标题：安全验证
副标题：为保障资金安全，请完成以下验证
字段：资金密码 + Google 验证码
按钮：取消 / 确认提交
```
