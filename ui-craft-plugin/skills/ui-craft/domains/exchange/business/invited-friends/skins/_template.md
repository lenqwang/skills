# Token Crew 皮肤模板

> 本模板用于创建 Token Crew 的节日/主题皮肤变体。
> 继承链：[_base.md](../_base.md) → [token-crew.md](../token-crew.md) → 本皮肤

---

## 区块 1: 元信息

```yaml
name: <skin-id>              # 文件名（不含 .md），如 spring-festival
description: <一句话描述>
theme: dark                  # 通常保持 dark
inherits: token-crew         # 固定继承 token-crew
active_period:               # 皮肤生效时间段（可选）
  start: YYYY-MM-DD
  end: YYYY-MM-DD
```
