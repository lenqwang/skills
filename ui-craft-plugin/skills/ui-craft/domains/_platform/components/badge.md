# Badge 徽标

> 在图标或内容上方附加少量数字或状态信息的提示组件，支持 5 种类型尺寸。

---

## Figma 链接

- [Badge_H5 Guidelines](https://www.figma.com/design/kQIx9xKLiyO6KZIbazNSM2/?node-id=8102-16370)

**组件 Token：** `BadgeV5-web` / `BadgeV5-H5`

---

## 类型与尺寸

| 类型 | Web 尺寸（宽 x 高） | H5 尺寸（宽 x 高） | 说明 |
|------|-------------------|-------------------|------|
| Dot（圆点） | 8 x 8 px | 8 x 8 px | 仅提示有新内容，无数字 |
| 1-9 | 16 x 16 px | 16 x 16 px | 单位数计数 |
| 10-99 | 22 x 16 px | 22 x 16 px | 两位数计数 |
| 99+ | 29 x 16 px | 29 x 16 px | 溢出计数 |
| New | 34 x 16 px | 30 x 12 px | 文字徽标 |

---

## 使用规范

- **Tag 类型**徽标（附文字时）：距文案**上间距 2 px**，**右间距 8 px**，居右对齐
- 文本类徽标建议限制在 **4-6 个英文字符**或 **2-4 个中文字符**以内
- Badge 叠加在图标右上角时，以图标边缘外 **2 px** 为基准偏移
