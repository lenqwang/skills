# Platform 布局

> 交易所 + Web3 共享的页面布局体系。与 Campaign 的"Hero + 区块"模式不同，
> Platform 采用"侧栏 + 主区 + 面板"的多面板布局。

---

## 布局模式

### platform.sidebar — 侧栏布局（默认）

最常见的交易所/Web3 页面布局，左侧导航 + 主区。

```
+------------------------------------------------------------------+
|  Header (56-64px)                                                |
+--------+---------------------------------------------------------+
|        |                                                         |
| Side   |  Main Content                                           |
| bar    |  +--------------------------------------------------+   |
| 240px  |  |  Page Header (标题 + 操作)                         |   |
|        |  +--------------------------------------------------+   |
| - Nav  |  |                                                  |   |
| - Menu |  |  Content Area                                    |   |
| - Quick|  |  (Table / Cards / Dashboard)                     |   |
|   Link |  |                                                  |   |
|        |  +--------------------------------------------------+   |
+--------+---------------------------------------------------------+
```

### platform.trading — 交易页布局（Exchange 专用）

多面板复杂布局，用于现货/合约交易页。

```
+------------------------------------------------------------------+
|  Header + MarketTicker                                           |
+----------+-----------------------------+-------------------------+
|          |                             |                         |
| Order    |  Chart (K线)                |  Trade                  |
| Book     |  (flex: 1)                  |  Panel                  |
| 280px    |                             |  280-320px              |
|          |                             |                         |
|          +-----------------------------+                         |
|          |  Open Orders / History      |                         |
|          |  (Tab 切换)                  |                         |
+----------+-----------------------------+-------------------------+
```

### platform.wallet — 钱包/资产布局（Web3 专用）

适用于钱包管理、资产概览等页面。

```
+------------------------------------------------------------------+
|  Header / Navbar                                                 |
+------------------------------------------------------------------+
|  Total Assets (总资产概览)                                        |
|  +--------------------------------------------------------------+|
|  |  ¥ 123,456.78          [显示/隐藏] [充值] [提现] [转账]       ||
|  +--------------------------------------------------------------+|
+------------------------------------------------------------------+
|  Account Tabs (账户 Tab)                                         |
|  +--------------------------------------------------------------+|
|  |  现货 | 合约 | 理财 | 量化 | Web3 | ...                      ||
|  +--------------------------------------------------------------+|
+------------------------------------------------------------------+
|  Asset List / Detail                                             |
|  +--------------------------------------------------------------+|
|  |  CoinTitle + Balance + Value + Actions                       ||
|  |  ...                                                         ||
|  +--------------------------------------------------------------+|
+------------------------------------------------------------------+
```

### platform.list — 列表页布局

用于行情列表、挂单记录等表格为主的页面。

```
+------------------------------------------------------------------+
|  Header                                                          |
+------------------------------------------------------------------+
|  Filter Bar (搜索 + 筛选 + 排序)                                 |
+------------------------------------------------------------------+
|  Tab Bar (分类 Tab)                                              |
|  +--------------------------------------------------------------+|
|  | 全部 | 现货 | 合约 | 杠杆 | 跟单 | ...                      ||
|  +--------------------------------------------------------------+|
+------------------------------------------------------------------+
|  Table / List                                                    |
|  +--------------------------------------------------------------+|
|  |  (Table V5 或 虚拟滚动列表)                                   ||
|  |  ...                                                         ||
|  +--------------------------------------------------------------+|
+------------------------------------------------------------------+
|  Pagination (Web) / Infinite Scroll (App)                        |
+------------------------------------------------------------------+
```

## 布局 Token

| Token | Web | App/H5 | 说明 |
|-------|-----|--------|------|
| `--sidebar-width` | 240px | 不适用 | 左侧栏宽度 |
| `--panel-width` | 280-320px | 全宽 | 交易面板宽度 |
| `--panel-gap` | 8px | 0 | 面板间距 |
| `--header-height` | 56-64px | 44-48px | 顶栏高度 |
| `--page-padding` | 24px | 16px | 页面内边距 |
| `--content-max-width` | 1440px | 不适用 | 内容最大宽度 |

## 平台差异

### Web 端

- 侧栏固定左侧，可折叠
- 多面板自由组合（Trading 布局最多 4 个面板）
- 鼠标 hover 交互丰富
- 最小宽度 1200px（Trading 页）

### H5/App 端

- 无侧栏，底部 Tab 切换
- 单面板占全屏，上下滑动
- 面板间通过页面跳转或 Sheet 弹出
- 手势操作（滑动、长按、双指缩放）

## 引用方式

domain 文件中声明布局：
- `layout: platform.sidebar` — 侧栏页面
- `layout: platform.trading` — 交易页（Exchange）
- `layout: platform.wallet` — 钱包页（Web3）
- `layout: platform.list` — 列表页
