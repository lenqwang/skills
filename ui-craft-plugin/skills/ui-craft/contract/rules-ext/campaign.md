# Campaign 扩展规则

> 活动线专属规则，继承 `contract/rules.md` 的 R1-R28 全局规则。

---

## R-C1: 活动页卡片交替 [W]

```
同一活动区块内：
外层用黑卡（规则描述）
内嵌白卡（奖励表格）
蓝卡用于推广/加码区块
```

**check**: 活动页场景 -> 检查卡片嵌套是否遵循黑卡 > 白卡的交替规律。蓝卡是否仅用于推广/加码。

**fix**: 交替不当 -> 调整卡片背景色使用，遵循黑卡外层、白卡内嵌的模式。

---

## R-C2: 活动页 Hero 必须包含倒计时或时间信息 [W]

```
活动页 Hero 区必须包含以下至少一项时间元素：
✅ 倒计时组件
✅ 活动时间范围文字
❌ 无任何时间信息
```

---

## R-C3: 活动页必须包含规则区块 [W]

```
每个活动页底部必须包含活动规则区块（rules-faq 组件）。
无规则区块 -> 违规。
```
