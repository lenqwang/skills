## 使用方式

通过 Figma MCP 拉取组件最新设计数据：

```bash
# 使用 MCP get_design_context 获取组件设计
# fileKey 和 nodeId 从本注册表查询
get_design_context(fileKey="kQIx9xKLiyO6KZIbazNSM2", nodeId="7065:53379")
```

通过 Figma REST API 批量获取（需要 Personal Access Token）：

```bash
curl -H "X-Figma-Token: <TOKEN>" \
  "https://api.figma.com/v1/files/<fileKey>/nodes?ids=<nodeId1>,<nodeId2>"
```
