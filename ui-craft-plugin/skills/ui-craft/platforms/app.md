# App 平台层

> App 端（iOS + Android）专属的设计约束和组件。
> 与 Web 端的差异约 30-35%，主要体现在触摸交互、反馈方式和导航模式。

## 平台特征

- 触摸交互为主（手势、滑动）
- 单屏/单任务布局
- 垂直导航（Navbar + Tabbar）
- 系统级反馈（Toast、ActionSheet）
- 安全区适配（StatusBar、HomeIndicator）
