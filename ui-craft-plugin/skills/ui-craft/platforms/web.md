# Web 平台层

> Web 端（桌面 + H5）专属的设计约束和组件。
> 与 App 端的差异约 30-35%，主要体现在导航模式、反馈方式和布局系统。

## 平台特征

- 鼠标交互为主（hover、右键菜单）
- 多窗口/多面板布局
- 水平导航（Header + LeftMenu）
- 键盘快捷键支持
- 响应式断点：768px / 1248px
